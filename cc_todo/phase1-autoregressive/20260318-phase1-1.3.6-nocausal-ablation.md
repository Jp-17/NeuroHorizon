# Phase 1.3.6 baseline_v2 non-causal ablation

## 1. 任务目标

在 baseline_v2 continuous 训练协议保持不变的前提下，只把 decoder self-attention 从因果掩码改成完全双向注意力，重跑：

- 250ms continuous
- 500ms continuous
- 1000ms continuous

并同时比较两种 bps 口径：

- `fp-bps`
- `ibl_mtm_bps`

## 2. 代码范围

- `examples/neurohorizon/configs/model/neurohorizon_small_{250ms,500ms}_noar.yaml`
- `examples/neurohorizon/configs/train_v2_nocausal_{250ms,500ms,1000ms}.yaml`
- `scripts/analysis/neurohorizon/run_phase1_v2_nocausal.sh`
- `scripts/analysis/neurohorizon/compare_phase1_v2_causal_ablation.py`

## 3. 输出目录

- 训练与评估：
  - `results/logs/phase1_v2_nocausal_250ms_cont/`
  - `results/logs/phase1_v2_nocausal_500ms_cont/`
  - `results/logs/phase1_v2_nocausal_1000ms_cont/`
- 对比：
  - `results/logs/phase1_v2_nocausal_comparison/`

## 4. 运行命令

```bash
conda activate poyo
cd /root/autodl-tmp/NeuroHorizon

bash scripts/analysis/neurohorizon/run_phase1_v2_nocausal.sh
python scripts/analysis/neurohorizon/compare_phase1_v2_causal_ablation.py
```
