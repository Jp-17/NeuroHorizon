## 1.1.7 评估指标补充

### 产出文件
- `torch_brain/utils/neurohorizon_metrics.py`：指标模块
- `scripts/analysis/neurohorizon/eval_psth.py`：PSTH-R² 独立评估脚本

### 实现内容
1. **fp-bps（Forward Prediction Bits Per Spike）**
   - 公式：`(NLL_null - NLL_model) / (N_spikes * ln2)`
   - Null model = 训练集 per-neuron 平均发放率
   - fp-bps = 0 → 模型与 null model 等效；> 0 → 优于；< 0 → 劣于
2. **per-bin fp-bps**：每个 time bin 独立计算，用于分析 AR 预测随时间步的衰减
3. **PSTH-R²**：Trial-averaged 神经活动 vs 预测的 R²，需 trial-aligned 数据加载
4. **辅助指标**：R²、Firing rate correlation、Poisson NLL（均已实现）
5. **工具函数**：
   - `compute_null_rates()`：遍历训练集计算 per-neuron 平均发放率
   - `build_null_rate_lookup()`：构建全局单元 → null rate 的查询表

## 1.1.8 AR 修复：预测反馈实现

### 产出文件
- `torch_brain/nn/prediction_feedback.py`：反馈编码模块（4 种方式）
- `torch_brain/nn/autoregressive_decoder.py`：集成方案 A（Query Augmentation）
- `torch_brain/models/neurohorizon.py`：forward() 传递 GT counts / generate() 使用预测反馈

### 实现内容

#### 已完成
- **TF≡AR 问题分析**：bin_query 仅由 learnable base + rotary time embed 构成，不携带前一步预测结果。Causal mask 仅限制隐状态可见性，不引入预测依赖。
- **方案 A 集成**：Query Augmentation，feedback 向量直接加到 bin_queries 上
- **4 种反馈编码方式**：
  | 方式 | 类 | 原理 |
  |------|-----|------|
  | mlp_pool | PerNeuronMLPPooling | MLP(concat(rate_emb, unit_emb)) → mean pool |
  | rate_weighted | RateWeightedSum | softmax(rates) 加权 unit_emb 求和 |
  | cross_attn | CrossAttentionPooling | 可学习 query attend to rate-augmented unit_emb |
  | none | NoFeedback | 返回零向量（对照组） |

#### 未完成
- **方案决策**：方案 B（预测记忆交叉注意力）和方案 C（重编码）未实现，需消融实验结果支持
- **编码方式决策**：4 种编码方式的效果对比需 1.2.3 验证后决定

## 1.1.9 Trial-Aligned 数据加载

### 产出文件
- `torch_brain/data/trial_sampler.py`：TrialAlignedSampler + TrialDatasetIndex
- `torch_brain/data/dataset.py`：get_trial_intervals() 方法

### 实现内容
1. **get_trial_intervals(split)**：从 HDF5 提取 {recording_id: {go_cue_time, target_id}}
2. **TrialAlignedSampler**：以 go_cue_time 为对齐点采样，每个 sample = 一个 trial
3. **TrialDatasetIndex**：继承 DatasetIndex，增加 target_id 和 go_cue_time 字段
4. **train.py 集成**：通过 trial_aligned config 开关选择 sampler

### 与连续窗口模式的关系
- 连续模式（RandomFixedWindowSampler）：随机位置、固定长度 → 训练
- Trial-aligned 模式（TrialAlignedSampler）：go_cue 对齐 → PSTH-R² 评估、方向调谐分析
